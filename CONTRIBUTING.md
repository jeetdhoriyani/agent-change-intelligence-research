# Contributing

## Evidence standard

Every factual field about a project should include a source URL and retrieval date. Avoid inferring adoption from stars alone. Separate verified facts from analyst judgment.

## Adding a project

1. Check `landscape/inclusion_criteria.md`.
2. Add one row to `landscape/projects.csv`.
3. Record primary sources in `source_urls`.
4. Add concise evidence notes for scored fields.
5. Run `python studies/validate_datasets.py`.

## Pull requests

Keep additions focused. Describe sources, classification decisions, and uncertainty.
